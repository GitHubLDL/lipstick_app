# lip_app

#### 介绍
使用uni-app编写的一款简单识别口红色号的程序，送给爱美的小姐妹，还有那些单纯的小兄弟~~~

#### 安装教程

1.  克隆/下载代码
2.  导入到 Hbuilder X
3.  点击运行，选择相应的模拟器。
